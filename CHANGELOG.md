# Changelog

## 0.1.0

Initial public open-source runtime release.

- Added transparent always-on-top desktop pet runtime.
- Added `.petpkg` import, validation, and package builder tooling.
- Added starter pet package and default bundled pet.
- Added customer delivery, order gate, seller handoff, readiness, and acceptance workflows.
- Added MIT license, public CI, contribution guide, security policy, and SignPath application notes.
