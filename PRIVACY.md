# MiraPet Privacy Policy

MiraPet is an open-source desktop pet runtime. The runtime stores pet configuration, imported pet package metadata, window position, scale, and diagnostic information on the user's local device.

## Local Data

MiraPet may store:

- app version
- app data path
- selected pet metadata
- window position and scale
- auto-start setting
- diagnostics generated by the user

## Customer Images

MiraPet support bundles intentionally exclude customer sprite image bytes, source reference images, and generated animation frames.

Custom pet production orders may include reference images and generated frames. Those files are handled outside the open-source runtime repository and are not committed to the public project.

## Support Bundles

When a user chooses to export diagnostics or a support bundle, they decide whether to send that file to the maintainer or service provider.

Support bundles may include app version, configuration, app data path, diagnostics, and pet manifest summaries.

## Third Parties

The open-source runtime does not require a hosted account service. Code signing for public Windows releases may use SignPath Foundation or another signing provider.
